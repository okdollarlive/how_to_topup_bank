For AYA, MAB & uab Bank

Visit nearest bank branch to make counter deposit for OK$ wallet topup. You have to provide with following information.

  1. OK$ Account Mobile Number
  2. NRC / ID
  3. Deposit Amount
  4. Depositor Name
  5. Depositor Contact Number
  6. Get a Bank Deposit Receipt.
    Check OK$ account number and other details mention on receipt.
    Keep the Receipt for record.
  7. Deposited Amount will be credited to OK$ account within (3) hours.
  8. Need to Refresh & Log in to OK$ account to check new wallet balance.
  9. If you do not receive OK$ amount please contact us.
  10. If you do not receive OK$ amount please contact us.

For CB & KBZ Bank

  1. To make deposit first need to generate " Bank Deposit Code ".
  2. Fill the data completely to generate " Bank Deposit Code ".
  3. Visit nearest bank branch with generated Code .
  4. Inform bank counter staff that you want to make deposit for OK$.
  5. Show the code & give amount to bank counter staff.
    (Amount as mentioned in code).
  6. After successful transaction the deposited amount will be credited to OK$ account within (1) hour.
  7. Need to Refresh & Log in to OK$ account to check new wallet balance.
  8. If you do not receive OK$ amount please contact us.
